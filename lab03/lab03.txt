Conclusiones.-

 - Odoo ofrece distintos módulos que se complementan entre si

 - El módulo de CRM de Odoo permite administrar nuestros contactos, así como iniciativas y posibles oportunidades y generar reportes de estas.

 - Mediante el módulo es posible guardar contactos como iniciativas, posteriormente convertirlas en oportunidades ganadas o perdidas.

 - Durante la instalación del módulo, Odoo nos ofrece una serie de pasos en los cuales podemos definir los puntos principales del negocio, con los cuales Odoo nos ayudará a cumplir los objetivos y expectativas planteadas.

 - El módulo CRM ofrece una interfaz intuitiva para la creación de clientes junto con datos personales que Odoo considera relevantes para la empresa. 

 - Es posible añadir productos, clientes, contactos, iniciativas desde cero o importar esa información desde otro sistema mediante archivos .csv

 - En la última versión de Odoo se permite la importación de datos mediante archivos .csv, en cambio no es posible la exportación.

 - Para la importación de contactos, estos deben estar correctamente ordenados en cuanto a las columnas que corresponden con el modelo actual. El módulo no ofrece un esquema para el ordenamiento de estas columnas.
